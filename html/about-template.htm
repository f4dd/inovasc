Introduction

INOVASC development goals were pure simplicity. No Java, PHP, databases or other enabling technology must be maintained. No cookies either. Just plain CGI technology written in 'C'.

The benefit is a centrally located scan client that keeps all results in one place. The design is kept so simple, even a non-IT person could start a scan.

Credits and Copyrights

INOVASC has been developed for use with a Apache webserver under Linux, connecting to a OpenVAS v4.x/v5.x server and is using libraries such as:

  • GNU libc
  • OpenSSL
  • CGIC
